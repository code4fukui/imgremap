# imgremap

A web-based tool for remapping and resizing images.

## Demo
https://code4fukui.github.io/imgremap/

## Features
- Allows user to specify 4 points on an image and remap the image to a new size based on those points
- Provides input fields for adjusting the coordinate values of the 4 points
- Resizes the output image to a specified width and height

## Usage
1. Drop an image file onto the page or click the canvas to select 4 points on the image.
2. Adjust the coordinate values of the 4 points as needed.
3. Set the desired width and height for the output image.
4. The remapped and resized image will be displayed in the second canvas.
