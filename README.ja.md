# imgremap

4点変換を用いて画像のパース（遠近感）を補正し、リマップ（再マッピング）するためのWebベースのツールです。

## デモ

https://code4fukui.github.io/imgremap/

2画面構成のインターフェースとなっています。左側のパネルに表示された元画像から4点を選択すると、右側のパネルに変換結果がリアルタイムで表示されます。

## 主な機能

-   **インタラクティブなポイント選択:** 元画像をクリックして、4つの頂点を指定します。
-   **パース補正:** 4点で指定した四角形の領域を、長方形にリマップ（変形）して出力します。
-   **座標による微調整:** 各ポイントのX/Y座標を手動で入力し、正確に調整できます。
-   **出力サイズの指定:** 出力画像の幅と高さを正確に指定できます。
-   **ライブプレビュー:** ポイントやサイズを調整すると、変換後の画像が即座に更新されます。
-   **ドラッグ＆ドロップ:** ページ上に画像ファイルをドラッグ＆ドロップするだけで、簡単に読み込めます。

## 使い方

1.  **画像の読み込み:** アプリケーションのウィンドウに画像ファイルをドラッグ＆ドロップするか、初期表示されているサンプル画像を使用します。
2.  **4点の選択:** 元画像（左側のキャンバス）をクリックし、以下の順序で4つのポイントを選択します。
    -   `p0`: 左上
    -   `p1`: 右上
    -   `p2`: 右下
    -   `p3`: 左下
3.  **微調整:**
    -   `p0` から `p3` の入力フィールドを使用して、座標を微調整します。
    -   右側のキャンバスの下にあるフィールドで、出力画像の `width`（幅）と `height`（高さ）を設定します。
4.  **結果の確認:** リマップされた画像が自動的に生成され、右側のキャンバスに表示されます。
