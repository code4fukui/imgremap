# imgremap

imgremap は、画像の座標をリマップする Webアプリケーションです。ユーザーが画像上の4点を指定すると、その座標に基づいて新しいサイズの画像を生成することができます。

## デモ
https://code4fukui.github.io/imgremap/

## 機能
- 画像をドラッグアンドドロップして表示する
- 画像上の4点の座標を指定する
- 指定した座標に基づいて新しいサイズの画像を生成する
- 生成した画像をキャンバスに表示する

## 使い方
1. 画像をドラッグアンドドロップしてアプリに表示します。
2. 画像上の4点の座標を入力フィールドに指定します。
3. 出力する画像のサイズを設定します。
4. 新しい画像がキャンバスに表示されます。
